# 🚗 Electric Vehicle Data Analysis Dashboard

![Dashboard Preview](Dashboard%201.png)

## 📌 Project Overview

This project presents an interactive **Electric Vehicle (EV) Data Analysis Dashboard** built using **Tableau**. It provides valuable insights into electric vehicle adoption, vehicle types, manufacturers, model trends, CAFV eligibility, and geographic distribution.

The dashboard enables users to explore EV market trends through interactive filters and visualizations, making it useful for business intelligence, market research, and policy analysis.

## 📈 Key Insights

- Total Vehicles Analyzed: **150,413**
- Battery Electric Vehicles (BEV): **116,745 (77.62%)**
- Plug-in Hybrid Electric Vehicles (PHEV): **33,668 (22.4%)**
- Average Electric Range: **67.83 Miles**
- Tesla dominates the EV market with over **54%** of total vehicles.
- Significant growth in EV registrations is observed after **2020**.
